# 🍕 Projeto MR.PIZZA - E-Commerce & UI/UX Design

Este repositório contém a implementação Frontend de uma plataforma de delivery (e-commerce) focada na otimização do fluxo de encomendas e na Experiência do Utilizador (UX). 

O grande destaque deste trabalho é demonstrar o ciclo completo de desenvolvimento moderno (Design-to-Code). Toda a interface e a jornada do utilizador foram inicialmente desenhadas e prototipadas no **Figma**, sendo posteriormente traduzidas para uma implementação *pixel-perfect* em código, garantindo total fidelidade visual e responsividade.

---

## 🛠️ Tecnologias e Arquitetura

Para inicializar o ambiente, configurar a base de dados e povoá-la automaticamente com dados de teste (seeding), abre o terminal na pasta raiz do projeto e executa os seguintes comandos:

* **Framework & Linguagem:** React, TypeScript, Vite
* **Styling & UI:** Tailwind CSS, componentes acessíveis (shadcn/ui)
* **Design & Prototipagem:** Figma

---

## 🚀 Como executar o projeto

Para inicializar o ambiente de desenvolvimento local com compilação rápida de assets (HMR via Vite), abre o terminal na pasta raiz do projeto e executa:

```bash
npm install
npm run dev
```
